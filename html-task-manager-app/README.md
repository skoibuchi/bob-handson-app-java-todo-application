# TaskManager - HTML版タスク管理アプリ

シンプルで使いやすいタスク管理Webアプリケーション（HTML/CSS/JavaScript版）

## 📋 概要

TaskManagerは、個人のタスク管理を支援するシンプルなWebアプリケーションです。
**HTML/CSS/JavaScript**で構築されており、追加のソフトウェアインストールは不要です。
ブラウザさえあれば、どのPCでもすぐに使い始めることができます。

このアプリケーションは**2つのバージョン**を提供しています：
- **単一ファイル版**: 配布が簡単（1ファイル）
- **分離版**: メンテナンスしやすい（HTML/CSS/JS分離）

### 特徴

✅ **インストール不要** - HTMLファイルをダブルクリックするだけ  
✅ **オフライン動作** - インターネット接続不要（初回のみCDN読み込みが必要）  
✅ **データ永続化** - ブラウザのLocalStorageにデータを保存  
✅ **レスポンシブデザイン** - PC・タブレット・スマホ対応  
✅ **カンバンボード** - 視覚的にタスクを管理

## 📱 主な機能

### タスク一覧（カンバンボード形式）

タスクをステータス別に4列で表示：

- **未着手（TODO）** - これから取り組むタスク
- **進行中（IN_PROGRESS）** - 現在作業中のタスク
- **保留（ON_HOLD）** - 一時的に保留中のタスク
- **完了（DONE）** - 完了したタスク

各タスクカードには以下の情報が表示されます：
- タイトル
- 説明（60文字まで）
- 優先度バッジ（高・中・低）
- 期限
- アクションボタン（詳細・編集）

### タスクの作成

1. 「新規作成」ボタンをクリック
2. 以下の情報を入力：
   - **タイトル**（必須、最大200文字）
   - **説明**（任意）
   - **ステータス**（未着手・進行中・保留・完了）
   - **優先度**（高・中・低）
   - **期限**（任意）
3. 「保存」ボタンをクリック

### タスクの表示

#### 一覧表示
- トップページで全タスクをカンバンボード形式で表示
- ステータス別に4列に分かれて表示
- 各列にタスク数が表示されます

#### 詳細表示
- タスクカードの「詳細」ボタンをクリック
- 以下の情報が表示されます：
  - タスクID
  - タイトル
  - 説明（全文）
  - ステータス
  - 優先度
  - 期限
  - 作成日時
  - 更新日時

### タスクの編集

**方法1: 一覧画面から**
1. タスクカードの「編集」ボタン（鉛筆アイコン）をクリック
2. 情報を編集
3. 「保存」ボタンをクリック

**方法2: 詳細画面から**
1. タスク詳細画面の「編集」ボタンをクリック
2. 情報を編集
3. 「保存」ボタンをクリック

### タスクの削除

1. タスク詳細画面の「削除」ボタンをクリック
2. 確認ダイアログで「OK」をクリック
3. タスクが削除され、一覧画面に戻ります

## 💾 データについて

### データの保存場所

タスクデータは**ブラウザのLocalStorage**に保存されます。

- **保存先**: ブラウザ内部のストレージ
- **容量**: 約5-10MB（ブラウザによる）
- **永続性**: ブラウザを閉じてもデータは保持されます

### 初期データ

初回起動時には、以下の5つのサンプルタスクが自動的に作成されます：

1. Spring Bootの学習（進行中、優先度：高）
2. データベース設計（完了、優先度：高）
3. 画面デザイン（未着手、優先度：中）
4. テスト実施（保留、優先度：中）
5. ドキュメント作成（未着手、優先度：低）

### データのリセット方法

データを初期状態に戻したい場合：

#### 方法1: ブラウザの開発者ツールを使用

1. ブラウザで `F12` キーを押して開発者ツールを開く
2. 「Application」タブ（Chromeの場合）または「ストレージ」タブ（Firefoxの場合）を選択
3. 左側のメニューから「Local Storage」を展開
4. 該当のURLを選択
5. `taskManagerData` を右クリックして削除
6. ページをリロード

#### 方法2: ブラウザのキャッシュをクリア

1. ブラウザの設定を開く
2. 「閲覧データの削除」または「キャッシュのクリア」を選択
3. 「Cookie とサイトデータ」を選択
4. 削除を実行
5. ページをリロード

## ⚠️ 注意事項

### データの保存について

- データは**ブラウザのLocalStorage**に保存されます
- **ブラウザのキャッシュをクリアするとデータが消えます**
- 定期的にバックアップを取ることをお勧めします（手動でメモなど）

### ブラウザについて

- **同じブラウザでのみデータが共有されます**
  - Chrome で作成したデータは Firefox では見えません
  - 同じブラウザの別ウィンドウでは共有されます
- **プライベートモード（シークレットモード）では使用できません**
  - プライベートモードではLocalStorageが無効化されるため

### セキュリティについて

- **認証機能はありません** - 単一ユーザーでの使用を想定
- **データは暗号化されていません** - 機密情報の保存は避けてください
- **ローカルのみで動作** - データは外部に送信されません

## 🔧 トラブルシューティング

### アプリが起動しない

**原因**: ブラウザが設定されていない

**解決方法**:
1. HTMLファイルを右クリック
2. 「プログラムから開く」を選択
3. ブラウザ（Chrome、Edge、Firefoxなど）を選択

### データが保存されない

**原因1**: プライベートモード（シークレットモード）で開いている

**解決方法**: 通常モードでブラウザを開き直してください

**原因2**: LocalStorageが無効化されている

**解決方法**: ブラウザの設定でCookieとサイトデータを有効にしてください

### 画面が正しく表示されない

**原因**: インターネット接続がない（初回起動時）

**解決方法**: 
- 初回起動時はBootstrapとBootstrap IconsをCDNから読み込むため、インターネット接続が必要です
- 一度読み込めば、ブラウザにキャッシュされるため、以降はオフラインでも動作します

### タスクが多すぎて重い

**原因**: 大量のタスクが保存されている

**解決方法**:
1. 不要なタスクを削除してください
2. または、データをリセットしてください
