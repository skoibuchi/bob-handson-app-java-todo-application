# TaskManager - 単一ファイル版

すべてのコード（HTML/CSS/JavaScript）を1つのファイルにまとめたバージョン

## 📁 ファイル構成

```
single-file/
├── task-manager.html   # すべてを含む単一ファイル
└── README.md          # このファイル
```

## 🚀 使い方

### 起動方法

1. `task-manager.html` をダブルクリック
2. デフォルトのブラウザで自動的に開きます
3. すぐに使用開始できます！

**または**

右クリック → 「プログラムから開く」 → お好みのブラウザを選択

## ✨ 特徴

### メリット

✅ **配布が超簡単**
- ファイル1つだけ
- メールに添付して送信可能
- USBメモリで簡単に持ち運び

✅ **セットアップ不要**
- ダブルクリックするだけ
- フォルダ構造を気にする必要なし
- 初心者でも簡単に使える

✅ **デモに最適**
- プレゼンテーション時に便利
- すぐに動作を見せられる
- 環境構築の説明が不要

### デメリット

⚠️ **メンテナンス性**
- コードが1つのファイルに集約されているため、編集がやや難しい
- 大規模な変更には不向き

⚠️ **デバッグ**
- ブラウザの開発者ツールで見ると、すべてのコードが1ファイルに表示される

## 🔧 カスタマイズ

### 注意事項

単一ファイルのため、編集時は以下に注意してください：

1. **バックアップを取る**
   - 編集前に必ずコピーを作成

2. **セクションを確認**
   - `<style>` タグ内: CSS
   - `<script>` タグ内: JavaScript
   - それ以外: HTML

3. **テストを忘れずに**
   - 編集後は必ず動作確認

### 簡単なカスタマイズ例

#### アプリ名の変更

```html
<!-- 変更前 -->
<title>TaskManager - タスク管理アプリ</title>

<!-- 変更後 -->
<title>MyTaskApp - 私のタスク管理</title>
```

```css

    background-color: #28a745 !important;  /* 緑色に変更 */
}
```

## 🐛 トラブルシューティング

### アプリが起動しない

**原因**: ブラウザが設定されていない

**解決方法**:
1. HTMLファイルを右クリック
2. 「プログラムから開く」を選択
3. ブラウザ（Chrome、Edge、Firefoxなど）を選択

### データが保存されない

**原因**: プライベートモード（シークレットモード）で開いている

**解決方法**: 通常モードでブラウザを開き直してください

### 画面が正しく表示されない

**原因**: インターネット接続がない（初回起動時）

**解決方法**: 
- 初回起動時はBootstrapとBootstrap IconsをCDNから読み込むため、インターネット接続が必要です
- 一度読み込めば、ブラウザにキャッシュされるため、以降はオフラインでも動作します
